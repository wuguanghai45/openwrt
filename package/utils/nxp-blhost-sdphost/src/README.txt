blhost 2.6.2

Command-line debug tool for sending individual commands to the bootloader. 

Platform supported
=====================
ROM Bootloader for LPC55Sxx, LPC55xx, RT5xx, RT6xx
MCUBOOT for LPC54S0xx, RT10xx, MK64F12, MK22F51212, MK65F18, MK66F18, MK80F25615, MK82F25615, MKE16, KM35Z7, MKV11Z7, MKV31F51212, MKV46F16, MKV58F24, MKW36Z4, MKW38Z4
